# textRPG

## 개발 언어 및 도구
- IDE : Visual Studio 2022
- 컴파일러 : C++17 Standard

## 빌드 주의사항

Beta: 세계 언어 지원을 위해 Unicode UTF-8 사용 활성화가 필요
- 활성화 방법 : 제어판 - 시계 및 국가 - 국가 또는 지역 - 관리자 옵션 탭 - 시스템 로캘 변경에서 활성화

## 프로젝트 구조 - 작성 중

TextRPG
 ┣ Source
 ┃ ┣ // 메인 함수 진입점
